# KRYPTOHUB
FINANCE DASHBOARD UI

This project is a frontend finance dashboard designed to present financial data in a structured and visually appealing way. It focuses on clean UI, layout design, and user experience.

FEATURES :
- Summary cards such as Total Balance, Income, and Expenses
- Display a list of transactions with details like Date, Amount
- Include basic features such as Simple filtering
- Show a few simple insights such as Highest spending category
- Clean design
- Includes Dark Mode

TECHNOLOGIES USED :
- HTML5
- CSS3
